# Agent Desk Skeleton

Public starter shell for a department, employee, workflow, and project desk UI.

This repository intentionally ships with no preconfigured departments, employees,
prompts, workflows, projects, logs, generated outputs, tokens, or local machine
paths. Add your own data locally after cloning.

## Run

```bash
npm install
npm start
```

Open:

```text
http://127.0.0.1:4173/
```

For the Electron shell:

```bash
npm run desktop
```
